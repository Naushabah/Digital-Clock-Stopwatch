# Digital Clock & Stopwatch

A digital clock and stopwatch built with HTML, CSS, and JavaScript as part of the CodTech IT Solutions Frontend Internship.

## Features
- Real-time digital clock with date and day display
- 12-hour format with AM/PM
- Stopwatch with Start, Pause, Resume and Reset
- Responsive design for all screen sizes

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
